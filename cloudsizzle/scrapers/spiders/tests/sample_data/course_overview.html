Course overview

Credits 5-8   cr
Teaching period I-III
Learning outcomes You learn to apply in a practical software project computer science and software engineering methods and tools that have been taught on other courses. You learn to evaluate the practical utility of the different methods and tools in various situations. You learn to work as a software developer in a large group.
Content Studying software engineering tools and practices in the context of
a software development project done as a team for a real customer.
The project includes project planning, requirements specification,
technical design, coding, quality assurance, system delivery and
producing documentation related to the previous activities. Course
participants generally work in activities related to the technical
implementation of the system.
Prerequisites T-76.3601 (mandatory), T-76.4602 (recommended), moderate
programming skills
Substitutes for courses T-76.4115 (8 cr) replaces former course T-76.115 Individual Project
Target audience
Assessment methods Project work.
Workload Lectures 10-15h Group work/independent work 27h * credits - lectures
Evaluation Based on the project work 1-5.
Study materials Instructions for the project work.
Language of instruction FI. Primarily Finnish. The assessed work may be completed in English upon request.
Course staff and contact information
Office hours
CEFR-level
Further information Students from the T-76.5115 course take the managerial roles in the projects. Lectures in Finnish, course materials in English.
